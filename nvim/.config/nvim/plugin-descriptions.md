# Neovim Plugin Descriptions

## LSP and Completion
- **LSP Config**: Mason + LSPConfig for language servers
- **Completions**: nvim-cmp with LuaSnip snippets

## Navigation and File Management
- **Telescope**: Fuzzy finder for files, buffers, grep
- **Neo-tree**: Tree-style file explorer with git info

## Editing and Dev Tools
- **Treesitter**: Syntax highlighting and parsing
- **Conform**: Formatter orchestration
- **Which-key**: Keybinding hints
- **nvim-tmux-navigation**: Move between Neovim and tmux splits

## Appearance
- **Catppuccin**: Colorscheme
- **mini.icons**: Lightweight devicons replacement
